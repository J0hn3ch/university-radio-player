Embedded player for university radio
====================================
Radio UniVersoMe embedded player written with Java, JavaFX and Spring
---------------------------------------------------------------------

**Note:** This application was made for Software Engineering project and it is not mantained.

# Requirements
* OpenJDK v16
* Eclipse IDE
* PostgreSQL
